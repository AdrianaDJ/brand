Wire symbol is standalone glyph derived from the logo lettering. It should be used as an icon to indicate Wire apps, preferrably with the full product name added in text. Additionally, the symbol can be used on its own if there is shortage of space and logo needs to be shortened or presented in a compact way.

Similarly to the logo, you should never misuse the symbol.

  • Don’t bend or twist
  • Don’t distort or stretch
  • Don’t rotate or tilt
  • Don’t use on a photo
  • Don’t use in multiple colors
  • Don’t apply effects

Wire symbol can be used as a freestanding element or within a surround. A common case for that is different shapes of app icons on software platforms or app stores. When placing the symbol inside a shape, you should adhere to platform design guidelines.

Regardless if the symbol is used as a standalone element or within a surrounding shape, it is recommended to use it simply in black or white. If it is necessary to use colored version, pick any of Wire palette colors.
