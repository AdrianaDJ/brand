Wire product user interfaces and website use standard system fonts. This provides the best support for language support, typographic features and Unicode compatibility. On Apple operating systems we use San Francisco, on Android Roboto; and on Microsoft ones Arial. If a manufacturer’s operating system has defined a different default font family, use that.

For other branding purposes, Wire uses ARS Maquette Pro — a versatile sans-serif with flexible typographic features.

The typeface was designed by Angus R. Shamal of ARS Type. For more information on usage and licensing, contact the foundry directly or get in touch with Wire Design.
